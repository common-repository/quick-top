A customizable button plugin for quickly scrolling to the top of the page.

== Description ==

Quick Top is a customizable plugin that allows you to add a "Back to Top" button to your WordPress site. Customize the button's appearance and behavior from the settings page.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/quick-top` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings->Quick Top screen to configure the plugin.
